
// Các kiểu dữ liệu trong Python: 

Số: int, float, complex

Chuỗi: str

Danh sách: list, tuple, range

Ánh xạ: dict

Tập hợp: set, frozenset

Logic: bool

None: NoneType 

Các toán tử trong Python

Số học: +, -, *, /, %, **, //

So sánh: ==, !=, >, <, >=, <=

Logic: and, or, not

Gán: =, +=, -=, *=, /=, %=, **=, //=

Bitwise: &, |, ^, ~, <<, >>

// Mệnh đề điều kiện và vòng lặp

Điều kiện: if, elif, else

Vòng lặp: for, while

// Kiểu dữ liệu True, False

Boolean: True, False
